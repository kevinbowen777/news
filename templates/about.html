About Page

This news application is an example of using the Django web framework to build a simple website. It supports user registration and verification. Once registered, users are able to create, edit, and delete posts.

This application also demonstrates the use of Bootstrap templates and Crispy Forms to create a pleasing, consistent visual design.

This project was originally based off an exercise from the excellent, introductory book written by Eric Matthes, "Python Crash Course". If you would like to obtain a copy of this software project, please visit the site: https://github.com/kevinbowen777/news .
